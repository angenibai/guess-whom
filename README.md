# Who Guessed ⁉️
Guess Who but with customisable characters

Play [here](https://who-guessed.web.app/)


## Running for development
#### Install dependencies
`npm i`

#### Start local server
`npm start`

## Running code in `scripts/`
#### Install dependencies
`pipenv install`

#### Activate shell
`pipenv shell`

#### Run scripts within the shell
`python example.py`

#### Close shell
`exit`
